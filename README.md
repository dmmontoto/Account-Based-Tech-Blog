# Account Based Tech Blog

![GitHub](https://img.shields.io/github/license/dmmontoto/Account-Based-Tech-Blog)

## Description

Welcome to the Account Based Tech Blog, a dynamic web platform where tech enthusiasts come together to share knowledge, ideas, and insights with the world. With user accounts for personalization, you can create and delete tech blog posts, engage in interactive discussions through comments, and manage accounts efficiently with the help of an SQL database. Powered by Node.js and Express.js, our project offers a seamless user experience with a responsive HTML, CSS, and JavaScript-based frontend. Join our vibrant tech community on Heroku, contribute to its growth, and stay updated with the latest tech trends.

## Usage

Once you are on the page, feel free to create an account on the secure database for the site. Once your account is created, please feel free to post whatever interesting technology information you have. Do not worry, the posts are on the database so your posts will be there for everyone to see everytime, unless you choose to delete the post.
